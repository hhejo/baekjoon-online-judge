# 1436 영화감독 숌



### 접근 방법

앞의 수가 0~5로 증가하고 그 다음 뒤의 수가 0 ~ 9로 증가하고 다시 앞.. 이 순서대로 했다가 잘 풀리지 않아 그냥 666부터 계속 더하고 종말의 숫자인지 확인했다.

### CODE

```python
N = int(input())
n, end = 0, 666
while True:
    if '666' in str(end):
        n += 1
    if N == n:
        break
    end += 1
print(end)
```

### 해설

`end`는 종말의 숫자이고 이 값을 계속 1씩 증가시킨다. `end` 안에 666이 있다면 종말의 숫자이므로 카운트 `n` 값을 1씩 증가시킨다. `n`이 `N`과 같아지면 종료한다.
