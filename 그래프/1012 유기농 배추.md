# 1012 유기농 배추



### 접근 방법

DFS나 BFS 아무거나 괜찮을 것 같았다. 좌표 하나씩 전체 탐색을 하는데 배추가 없는 땅이거나 방문했다면 탐색하지 않는다.

### CODE

```python
import sys

input = sys.stdin.readline


def dfs():
    global MAP, N, M
    di, dj = (-1, 0, 1, 0), (0, 1, 0, -1)
    visited = [[0] * M for _ in range(N)]
    cnt = 0
    for i in range(N):
        for j in range(M):
            if 0 == MAP[i][j] or visited[i][j]:
                continue
            s = [(i, j)]
            cnt += 1
            visited[i][j] = cnt
            while s:
                ci, cj = s.pop()
                for d in range(4):
                    ni, nj = ci + di[d], cj + dj[d]
                    if 0 <= ni < N and 0 <= nj < M and MAP[ni][nj] and not visited[ni][nj]:
                        s.append((ni, nj))
                        visited[ni][nj] = cnt
    return cnt


T = int(input())
for test_case in range(1, T + 1):
    M, N, K = map(int, input().split())
    MAP = [[0] * M for _ in range(N)]
    for _ in range(K):
        X, Y = map(int, input().split())
        MAP[Y][X] = 1
    ans = dfs()
    print(ans)
```

### 해설

시작 위치를 따로 주지 않아서 좀 헷갈렸다.
