## CODE 1

```python
import sys

input = sys.stdin.readline

n, k = map(int, input().split())

num = [True] * (n + 1)
cnt = 0
ans = 0
for i in range(2, n + 1):
    for j in range(i, n + 1, i):
        if num[j]:
            num[j] = False
            cnt += 1
            if cnt == k:
                ans = j

print(ans)
```
