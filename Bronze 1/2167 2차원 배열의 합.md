# 2167 2차원 배열의 합


## 1.

```python
N, M = map(int, input().split())
li = [[0] * (M + 1)] + [[0] + list(map(int, input().split())) for _ in range(N)]
K = int(input())
for _ in range(K):
    i, j, x, y = map(int, input().split())
    ans = 0
    for c in range(i, x + 1):
        for r in range(j, y + 1):
            ans += li[c][r]
    print(ans)
```

PyPy3로 통과함. DP를 써야 시간 초과가 나지 않는다고 한다.
