# 10163 색종이



```python
N = int(input())
li = [[0] * 1001 for _ in range(1001)]
for num in range(1, N + 1):
    X, Y, W, H = map(int, input().split())
    for x in range(X, X + W):
        for y in range(Y, Y + H):
            li[x][y] = num
for num in range(1, N + 1):
    print(sum(row.count(num) for row in li))
```
