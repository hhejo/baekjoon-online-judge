# 9012 괄호



```python
import sys

for _ in range(int(sys.stdin.readline())):
    stk = []
    for p in sys.stdin.readline().rstrip():
        if p == '(':
            stk.append(p)
        else:
            if stk:
                stk.pop()
            else:
                stk.append(p)  # 아래 if stk를 참으로 만들기 위해
                break
    print('NO') if stk else print('YES')
```
