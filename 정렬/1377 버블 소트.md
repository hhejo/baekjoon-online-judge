# 1377 버블 소트

---

## CODE

```python
import sys

input = sys.stdin.readline

N = int(input())
A = [(int(input()), i) for i in range(N)]


# 풀이
def solve():
    global N, A

    sorted_A = sorted(A)
    arr = [sorted_A[i][1] - i + 1 for i in range(N)]

    ans = max(arr)
    print(ans)
    return


solve()

```
