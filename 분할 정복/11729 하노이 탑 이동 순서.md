# 11729 하노이 탑 이동 순서

---

## CODE

```python
import sys

input = sys.stdin.readline

N = int(input())


# 풀이
def solve():
    global N

    def move(n, x, y):
        if n == 0:
            return
        move(n - 1, x, 6 - x - y)
        arr.append(f'{x} {y}')
        move(n - 1, 6 - x - y, y)
        return

    arr = []
    move(N, 1, 3)
    print(len(arr), *arr, sep='\n')
    return


solve()

```

## 해설

1. 1 ~ n-1 원판을 x에서 z로 이동 (z는 x와 y가 아닌 원판)
2. n 원판을 x에서 y로 이동
3. 1 ~ n-1 원판을 z에서 y로 이동
