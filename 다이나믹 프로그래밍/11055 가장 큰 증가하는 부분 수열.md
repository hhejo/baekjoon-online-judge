# 11055 가장 큰 증가하는 부분 수열

---

## CODE

```python
import sys

input = sys.stdin.readline

N = int(input())  # 크기
A = list(map(int, input().split()))  # 수열


# 풀이
def solve():
    global N, A
    dp = list(A)

    # 0 ~ N-1
    for i in range(N):
        # 0 ~ i-1
        for j in range(i):
            if A[j] < A[i] and dp[j] + A[i] > dp[i]:
                dp[i] = dp[j] + A[i]  # 갱신

    ans = max(dp)  # 이 중에서 가장 큰 값
    print(ans)  # 정답 출력
    return


solve()

```

## 해설

`31256KB`, `80ms`
