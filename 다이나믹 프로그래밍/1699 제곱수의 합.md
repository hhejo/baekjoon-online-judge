# 1699 제곱수의 합

---

## CODE 1

```python
import sys

input = sys.stdin.readline

N = int(input())


# 풀이
def solve():
    global N
    dp = [i for i in range(N + 1)]

    for i in range(1, N + 1):
        for j in range(1, int(i ** 0.5) + 1):
            if dp[i] > dp[i - j * j] + 1:
                dp[i] = dp[i - j * j] + 1

    ans = dp[N]
    print(ans)
    return


solve()

```

## 해설 1

`35108KB`, `4884ms`
