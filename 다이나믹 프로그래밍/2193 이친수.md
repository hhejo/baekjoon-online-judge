## CODE 1

```python
import sys

input = sys.stdin.readline

# 입력
N = int(input())


# 풀이
def solve():
    global N
    dp = [0, 1, 1] + [0] * 89

    # n 자리 이친수 = (n - 1 자리 이친수 + '0') + (n - 2 자리 이친수 + '01')
    for i in range(3, N + 1):
        dp[i] = dp[i - 2] + dp[i - 1]

    print(dp[N])
    return


solve()

```

## 해설 1

n 자리 이친수는 n - 1자리 이친수 맨 뒷자리에 0을 붙인 이친수의 개수와

n - 2자리 이친수에 01을 붙인 이친수의 개수를 더하면 된다.

1차원 배열 Bottom-Up 방식으로 풀었다.

`31256KB`, `40ms`

## CODE 2

```python
import sys

input = sys.stdin.readline

# 입력
N = int(input())


# 풀이
def solve():
    global N
    dp = [[0, 0] for _ in range(91)]

    # n 자리 이친수 = (n - 1 자리 이친수 + '0') + (n - 1 자리 이친수 + '1')
    dp[1][0], dp[1][1] = 0, 1
    dp[2][0], dp[2][1] = 1, 0
    for i in range(3, N + 1):
        dp[i][0] = dp[i - 1][0] + dp[i - 1][1]
        dp[i][1] = dp[i - 1][0]

    print(sum(dp[N]))  # 정답 출력
    return


solve()

```

## 해설 2

2차원 배열을 이용해서 풀었다.

`dp[n][k]`는 맨 뒤가 `k`로 끝나는 n자리 이친수 개수를 의미한다.

`31256KB`, `44ms`

## CODE 3

```python
import sys

input = sys.stdin.readline

# 입력
N = int(input())


# 풀이
def solve():
    global N
    dp = [0] * 91

    def f(n):
        if n == 1 or n == 2:
            return 1
        if dp[n] > 0:
            return dp[n]
        dp[n] = f(n - 2) + f(n - 1)
        return dp[n]

    ans = f(N)
    print(ans)  # 정답 출력
    return


solve()

```

## 해설 3

Top-Down 방식

`31256KB`, `40ms`
