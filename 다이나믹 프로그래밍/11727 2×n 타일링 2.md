## CODE 1

```python
import sys

input = sys.stdin.readline

# 입력
N = int(input())


# 풀이
def solve():
    global N
    MOD = 10007
    dp = [1, 1] + [0] * 999

    for i in range(2, N + 1):
        dp[i] = (dp[i - 2] * 2 + dp[i - 1]) % MOD

    ans = dp[N]
    print(ans)
    return


solve()

```

Bottom-Up 방식 DP

## CODE 2

```python
import sys

input = sys.stdin.readline

# 입력
N = int(input())


# 풀이
def solve():
    global N
    MOD = 10007
    dp = [0] * (N + 1)

    def calc(n):
        if n == 0 or n == 1:
            return 1
        if dp[n] > 0:
            return dp[n]
        dp[n] = (calc(n - 2) * 2 + calc(n - 1)) % MOD
        return dp[n]

    ans = calc(N)
    print(ans)
    return


solve()

```

## 해설 2

Top-Down 방식 DP (재귀)

`31316KB`, `44ms`
