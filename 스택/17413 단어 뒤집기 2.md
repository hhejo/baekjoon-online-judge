## CODE

```python
ans = ''
S = input().replace('>', '<').split('<')
for i in range(len(S)):
    if i % 2:
        ans += '<' + S[i] + '>'
    else:
        ans += ' '.join(s[::-1] for s in S[i].split())
print(ans)
```
