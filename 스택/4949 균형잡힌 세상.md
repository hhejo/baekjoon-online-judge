## CODE

```python
import sys

while True:
    string = sys.stdin.readline().rstrip()
    if string == '.':
        break

    stk, ans = [], 'yes'
    pair = {')': '(', ']': '['}
    for s in string:
        if s in '([':
            stk.append(s)
        elif s in ')]':
            if stk and stk[-1] == pair[s]:
                stk.pop()
            else:
                ans = 'no'
                break
    if stk:
        ans = 'no'
    print(ans)
```
