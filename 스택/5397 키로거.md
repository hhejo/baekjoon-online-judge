# 5397 키로거

---

### CODE

```python
for TC in range(1, int(input()) + 1):
    S = input()
    left, right = [], []
    for s in S:
        if '<' == s and left:
            right.append(left.pop())
        elif '>' == s and right:
            left.append(right.pop())
        elif '-' == s and left:
            left.pop()
        elif s not in '<>-':
            left.append(s)
    print(''.join(left + right[::-1]))
```
