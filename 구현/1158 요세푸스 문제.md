## CODE 1

```python
import sys

input = sys.stdin.readline

n, k = map(int, input().split())

arr = list(range(1, n + 1))

res = []
idx = 0
for _ in range(n):
    idx += k - 1
    if idx >= len(arr):
        idx %= len(arr)
    res.append(arr.pop(idx))

print('<', ', '.join(map(str, res)), '>', sep='')
```
