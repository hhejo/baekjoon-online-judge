## CODE 1

```python
import sys

input = sys.stdin.readline

n = int(input())
a = [input().strip() for _ in range(n)]

ans = len(a)
for word in a:
    for i in range(len(word) - 1):
        if word[i] != word[i + 1]:
            if word[i] in word[i+1:]:
                ans -= 1
                break

print(ans)
```
