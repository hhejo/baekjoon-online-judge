## CODE

```python
import sys

input = sys.stdin.readline

M = int(input())
s = 0
for _ in range(M):
    C = input().rstrip().split()
    cmd = C[0]
    x = 0
    if cmd == 'all' or cmd == 'empty':
        pass
    else:
        x = int(C[1])

    if cmd == 'add':
        s = s | (1 << x)
    elif cmd == 'remove':
        s = s & ~(1 << x)
    elif cmd == 'check':
        if s & (1 << x):
            print(1)
        else:
            print(0)
    elif cmd == 'toggle':
        s = s ^ (1 << x)
    elif cmd == 'all':
        s = (1 << 21) - 1
    elif cmd == 'empty':
        s = 0

```

## 해설

all 명령어에서 20이 아니라 21이다
