## CODE 1

```python
import sys

input = sys.stdin.readline

n = int(input())
a = [tuple(map(int, input().split())) for _ in range(n)]

g = [[0] * 100 for _ in range(100)]

for rect in a:
    for x in range(rect[1], rect[1] + 10):
        for y in range(rect[0], rect[0] + 10):
            g[x][y] = 1

ans = sum(row.count(1) for row in g)

print(ans)
```
