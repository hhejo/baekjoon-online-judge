# 2309 일곱 난쟁이



## 1.

```python
ARR = [0] * 9
for i in range(9):
    ARR[i] = int(input())
tmp = []
for i in range(8):
    for j in range(i + 1, 9):
        if 100 == sum(ARR) - ARR[i] - ARR[j]:
            tmp = [ARR[i], ARR[j]]
            break
    if tmp:
        break
ans = sorted(set(ARR) - set(tmp))
print(*ans, sep='\n')
```



## 2.

```python
from itertools import combinations

ARR = [int(input()) for _ in range(9)]
ans = []
for x, y in combinations(ARR, 2):
    if 100 == sum(ARR) - x - y:
        ans = sorted(set(ARR) - {x, y})
        break
print(*ans, sep='\n')
```
