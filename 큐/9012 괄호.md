### CODE

```python
import sys

for _ in range(int(sys.stdin.readline())):
    stk = []
    parenthesis_string = sys.stdin.readline().rstrip()  # .rstrip() 없으면 공백 문자 때문에 틀림
    for ps in parenthesis_string:
        if ps == '(':
            stk.append(ps)
        else:
            if stk:
                stk.pop()
            else:
                stk.append(ps)  # 아래 if else 에서 NO 출력을 위해 append. 다른 문자를 넣어도 상관 없음
                break
    if stk:
        print('NO')
    else:
        print('YES')
```
