## CODE 1

```python
import sys

input = sys.stdin.readline

n = int(input())
arr = [tuple(map(int, input().split())) for _ in range(n)]
arr.sort(key=lambda x: x[0])

check = 0
for arrival, takes in arr:
    if check < arrival:
        check = arrival
    check += takes

print(check)
```
