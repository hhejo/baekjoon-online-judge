## CODE 1

```python
import sys

input = sys.stdin.readline

s = input().strip()

cnt, prev = 0, 'A'
for ch in s:
    left = ord(ch) - ord(prev)
    if left < 0:
        left += 26
    right = ord(prev) - ord(ch)
    if right < 0:
        right += 26
    cnt += min(left, right)
    prev = ch

print(cnt)
```
