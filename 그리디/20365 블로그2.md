## CODE 1

```python
import sys

input = sys.stdin.readline

n = int(input())
s = input().strip()

colors = {'B': 0, 'R': 0}
colors[s[0]] += 1
for i in range(1, n):
    if s[i - 1] != s[i]:
        colors[s[i]] += 1

ans = min(colors['B'], colors['R']) + 1
print(ans)
```
