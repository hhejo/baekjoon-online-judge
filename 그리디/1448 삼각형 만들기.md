## CODE 1

```python
import sys

input = sys.stdin.readline

n = int(input())
arr = [int(input()) for _ in range(n)]
arr.sort(reverse=True)

ans = -1
for i in range(n - 2):
    if arr[i] < arr[i + 1] + arr[i + 2]:
        ans = arr[i] + arr[i + 1] + arr[i + 2]
        break

print(ans)
```
