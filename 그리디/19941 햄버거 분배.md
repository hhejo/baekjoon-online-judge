## CODE

```python
import sys

input = sys.stdin.readline

#

N, K = map(int, input().split())
a = list(input().rstrip())
ans = 0
for i in range(N):
    if a[i] == 'P':
        for j in range(max(i - K, 0), min(i + K + 1, N)):
            if a[j] == 'H':
                a[j] = 0
                ans += 1
                break
print(ans)

```
