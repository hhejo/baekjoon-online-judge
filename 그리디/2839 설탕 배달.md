## CODE 2

Greedy, Python

```python
import sys

N = int(sys.stdin.readline())

n, cnt = N, 0

while n % 5:
    n -= 3
    cnt += 1

if n < 0:
    cnt = -1
else:
    cnt += n // 5

print(cnt)
```
