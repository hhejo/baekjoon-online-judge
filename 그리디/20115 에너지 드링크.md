## CODE 1

```python
import sys

input = sys.stdin.readline

#

N = int(input())
a = sorted(map(int, input().split()), reverse=True)

acc = a[0]
for i in range(1, N):
    acc += a[i] / 2

print(f'{acc:.1f}')

```
