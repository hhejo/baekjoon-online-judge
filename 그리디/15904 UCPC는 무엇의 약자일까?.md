## CODE 1

```python
import sys

input = sys.stdin.readline

s = input().strip()

cnt = start = 0
for ucpc in 'UCPC':
    for i in range(start, len(s)):
        if ucpc == s[i]:
            cnt += 1
            start = i + 1
            break

ans = 'I hate UCPC'
if cnt == 4:
    ans = 'I love UCPC'
print(ans)
```
