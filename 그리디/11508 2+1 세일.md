## CODE 1

```python
import sys

input = sys.stdin.readline

n = int(input())
arr = [int(input()) for _ in range(n)]
arr.sort(reverse=True)

acc = 0
for i in range(0, (n // 3) * 3, 3):
    acc += arr[i] + arr[i + 1]
if n % 3 == 2:
    acc += arr[n - 2] + arr[n - 1]
elif n % 3 == 1:
    acc += arr[n - 1]

print(acc)
```
