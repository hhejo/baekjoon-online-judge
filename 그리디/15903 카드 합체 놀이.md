## CODE 1

```python
import sys
from heapq import heapify, heappush, heappop

input = sys.stdin.readline

n, m = map(int, input().split())
arr = list(map(int, input().split()))

heapify(arr)
for _ in range(m):
    x = heappop(arr)
    y = heappop(arr)
    heappush(arr, x + y)
    heappush(arr, x + y)
ans = sum(arr)
print(ans)
```
