## CODE 1

```python
import sys

input = sys.stdin.readline

n, m = map(int, input().split())
j = int(input())
arr = [int(input()) for _ in range(j)]

left = 1
right = m
acc = 0
for x in arr:
    if x < left:
        dist = left - x
        acc += dist
        right -= dist
        left = x
    elif right < x:
        dist = x - right
        acc += dist
        left += dist
        right = x
print(acc)
```
