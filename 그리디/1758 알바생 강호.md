## CODE 1

```python
import sys

input = sys.stdin.readline

n = int(input())
arr = sorted([int(input()) for _ in range(n)], reverse=True)

acc = 0
for i in range(n):
    res = arr[i] - ((i + 1) - 1)
    if res < 0:
        res = 0
    acc += res

print(acc)
```
