## CODE 1

```python
import sys

input = sys.stdin.readline

test_case = 0
while True:
    l, p, v = map(int, input().split())
    if l == p == v == 0:
        break
    test_case += 1
    ans = l * (v // p) + min((v % p), l)
    print(f'Case {test_case}: {ans}')
```
