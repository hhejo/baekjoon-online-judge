## CODE 1

```python
import sys

input = sys.stdin.readline

n, m = map(int, input().split())
p = [int(input()) for _ in range(m)]
p.sort()

max_price = max_earn = 0
for i in range(m):
    egg = min(n, m - i)
    price, earn = p[i], p[i] * egg
    if max_earn < earn:
        max_price, max_earn = price, earn

print(max_price, max_earn)
```
