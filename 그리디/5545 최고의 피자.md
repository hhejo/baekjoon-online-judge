## CODE 1

```python
import sys

input = sys.stdin.readline

n = int(input())
a, b = map(int, input().split())
c = int(input())
d = [int(input()) for _ in range(n)]
d.sort(reverse=True)

price = a
calories = c
maxx = calories / price
for i in range(n):
    price += b
    calories += d[i]
    maxx = max(maxx, calories / price)

ans = int(maxx)
print(ans)
```
