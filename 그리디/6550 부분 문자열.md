## CODE 1

```python
import sys

input = sys.stdin.readline

while True:
    try:
        s, t = input().split()
        idx = 0
        for i in range(len(t)):
            if t[i] == s[idx]:
                idx += 1
                if idx == len(s):
                    break
                continue
        ans = 'Yes'
        if idx != len(s):
            ans = 'No'
        print(ans)
    except:
        break
```
