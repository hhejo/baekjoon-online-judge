## CODE 1

```python
import sys

input = sys.stdin.readline

for test_case in range(int(input())):
    j, n = map(int, input().split())
    a = [tuple(map(int, input().split())) for _ in range(n)]
    a.sort(key=lambda x: -(x[0] * x[1]))
    candy, ans = j, 0
    for i in range(n):
        candy -= a[i][0] * a[i][1]
        if candy <= 0:
            ans = i + 1
            break
    print(ans)
```
