## CODE 1

```python
import sys

input = sys.stdin.readline

a, b = map(int, input().split())
n = int(input())
arr = [int(input()) for _ in range(n)]

cnt = abs(a - b)
for x in arr:
    cnt = min(cnt, abs(x - b) + 1)

print(cnt)
```
